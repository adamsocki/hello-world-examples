Hello World in Various Languages

Here are some "Hello World" examples in different programming languages:

C:
#include <stdio.h>
int main() {
  printf("Hello, World!\n");
  return 0;
}
        
C++:
#include <iostream>
int main() {
  std::cout << "Hello, World!" << std::endl;
  return 0;
}
        
Java:
public class Main {
  public static void main(String[] args) {
    System.out.println("Hello, World!");
  }
}
        
Python:
print("Hello, World!")
        
JavaScript:
console.log("Hello, World!");
        
Ruby:
puts "Hello, World!"
        
Go:
package main
import "fmt"
func main() {
  fmt.Println("Hello, World!")
}
        
Swift:
print("Hello, World!")
        
Kotlin:
fun main() {
  println("Hello, World!")
}
        
Rust:
fn main() {
    println!("Hello, World!");
}
